# Time Management Frontend

The frontend of the time management app written in Javascript/React.

## Requirements

- Node 14.5.0
- React 16.13.1
- Yarn 1.12.4

## Installation

- `yarn`
- `yarn start`

The app will run on `http://localhost:3000/`

## Tests

- Unit test with Jest: `yarn jest`
- E2E test with Cypress: `yarn cypress`
